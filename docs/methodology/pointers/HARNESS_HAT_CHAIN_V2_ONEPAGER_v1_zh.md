## 一句话

工作区 Harness V2 帽链 = **00 编排 + 10 双轨思考 + 20 双类型书面审 + 人闸 + 30–40–可选 50**，目标是把 AI 编码做成 **有规格冻结点、有书面签收、有命令证据、有独立复检、有 CI 终审** 的 PR 级交付。

## 关键纪律

- **双 20**：`20-spec-audit` 审 SPEC，`20-task-audit` 审 task；**不可合并为单审**。  
- **人闸**：`HG-AUDIT-R1` 必须在 30 执行前书面 approved；00/Agent **禁止代签**。  
- **50 可选**：非每个 task 都须 50，但 40 自检证据必须可复核。  
- **invoke ≠ cache**：invoke 是 **开帽快照**，不是 prompt 缓存。
