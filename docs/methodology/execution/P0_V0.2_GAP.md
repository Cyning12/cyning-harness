## 3. P0 演示命令（复制即用）

```bash
# 0. 准备空业务仓
mkdir -p /tmp/harness-p0-demo && cd /tmp/harness-p0-demo
git init -q

# 1. 安装（harness-only）
CYNING_HARNESS=/path/to/cyning-harness \
  "$CYNING_HARNESS/wizard/install.sh" --target . --preset harness-only --ide cursor,agents

# 2. 复制金样 task
cp "$CYNING_HARNESS/examples/demo_checkout/task_demo_p0_golden_v1.md" \
   docs/tasks/active/

# 3. sync 纪律层
CYNING_HARNESS=/path/to/cyning-harness \
  "$CYNING_HARNESS/wizard/harness-sync.sh" plan --target .
CYNING_HARNESS=/path/to/cyning-harness \
  "$CYNING_HARNESS/wizard/harness-sync.sh" apply --target .

# 4. gate-check（签 HG-AUDIT-R1 前 → exit 2）
CYNING_HARNESS=/path/to/cyning-harness \
  "$CYNING_HARNESS/wizard/gate-check.sh" --target . \
  --task docs/tasks/active/task_demo_p0_golden_v1.md

# 5. IDE：@ task + prompts/10 → 22 →（人签 HG-AUDIT-R1）→ 30
# 详见 examples/demo_checkout/ACCEPTANCE.md
```
